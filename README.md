# Palindrome Number Checker in Java

## Overview

This project contains a small Java program that checks whether a number entered by the user is a **palindrome** or not.

A palindrome number is a number that remains the same when its digits are reversed.

Example:

* 121 → Palindrome
* 1331 → Palindrome
* 245 → Not a palindrome

The program reverses the digits of the number and compares the reversed value with the original number.

---

## Program Logic

The program follows these steps:

1. Ask the user to enter a number.
2. Store the original value of the number.
3. Extract the last digit using the modulus (`%`) operator.
4. Build the reversed number digit by digit.
5. Continue the process until the number becomes zero.
6. Compare the reversed number with the original number.
7. Display whether the number is a palindrome or not.

---



## Example Execution

Input

```
Enter a number:
1331
```

Output

```
The number is a palindrome.
```
